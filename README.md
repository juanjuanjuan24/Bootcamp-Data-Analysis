# 📊 Data Analysis Bootcamp Repository

## 📌 Overview

This repository contains practical work developed during a data analysis bootcamp, focused on building foundational skills in Python for data exploration and problem understanding.

The project is currently in an early development stage and emphasizes exploratory thinking rather than fully structured data workflows.

---

## 🎯 Objective

The main goal of this repository is to document the learning process and the development of core data analysis skills, including:

* Understanding data structures in Python
* Developing logical thinking for data exploration
* Performing initial exploratory analysis (EDA)

This repository is intended as both a personal learning record and a foundation for future, more advanced projects.

---

## 🛠️ Tools & Environment

* Python
* Google Colab

At this stage, the work focuses on core Python functionality without the use of specialized data analysis libraries.

---

## 📊 Current Scope of Work

* Basic data exploration techniques
* Logical analysis of simple datasets or structured inputs
* Step-by-step problem solving using Python

> ⚠️ Note: Advanced tools such as Pandas, NumPy, or data visualization libraries are not yet implemented in this repository.

---

## 🚧 Project Status

**In Development**

This repository is continuously evolving as new concepts are introduced throughout the bootcamp. Future updates are expected to include:

* Use of data analysis libraries (e.g., Pandas, NumPy)
* Data visualization techniques
* More complex datasets and case studies

---

## ▶️ How to Use

1. Open the notebooks in Google Colab
2. Run each cell sequentially
3. Modify inputs and experiment with the code to reinforce learning

---

## 📚 Learning Focus

Throughout this project, the main areas of learning include:

* Writing clean and understandable Python code
* Structuring logical solutions to problems
* Developing a mindset for data analysis

---

## 🔍 Limitations

This repository does **not yet** include:

* Real-world datasets
* Advanced data processing libraries
* Production-level analysis workflows

These elements will be incorporated in later stages of the bootcamp.
